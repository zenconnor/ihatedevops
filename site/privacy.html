Privacy policy

Last updated: July 21, 2026

The short version

No ads, no accounts, no forms, and we sell nothing about you. We use Google Analytics to see aggregate page stats so we know whether anyone reads this thing.

What we collect

  • Aggregate usage via Google Analytics: pages viewed, referrer, rough geography, device type, and clicks on the install/GitHub buttons. Analytics sets cookies; block them with any content blocker or Google's opt-out and the site works identically.
  • Our hosting provider may keep standard server logs (IP address, user agent, request time) for security and operations. We don't access or use these beyond that.

The install script

The install command downloads files from GitHub and copies one folder to ~/.claude/skills/ on your machine. It sends nothing anywhere, phones home to no one, and has no telemetry. It's ~30 lines of shell — read it yourself.

Third parties

Downloads are served by GitHub, which has its own privacy statement.

Changes & contact

If this policy changes, the update lands in the git history like everything else. Questions: open an issue on the repo.
